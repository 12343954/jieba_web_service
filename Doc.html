Jieba Webapi Documentation

  • /api/cut/<string:str_words>

    分词：HMM=False 模式

  • /api/suggest_freq/<string:str_words>

    添加生词，str_words 生词要用英文逗号（,）拼接

    print('/'.join(jieba.cut('「台中」正确应该不会被切开', HMM=False)))
#「/台/中/」/正确/应该/不会/被/切开
print(jieba.suggest_freq('台中', True))
#69
print('/'.join(jieba.cut('「台中」正确应该不会被切开', HMM=False)))
#「/台中/」/正确/应该/不会/被/切开

  • /api/suggest_split/<string:str_words>

    建议拆分，str_words 要拆分的词组用英文逗号（,）拼接

    print('/'.join(jieba.cut('如果放到post中将出错。', HMM=False)))
#如果/放到/post/中将/出错/。
print(jieba.suggest_freq(('中', '将'), True))
#494
print('/'.join(jieba.cut('如果放到post中将出错。', HMM=False)))
#如果/放到/post/中/将/出错/。
